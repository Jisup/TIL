# Collection Class

### HashSet<type>
HashSet 사용방법!

>생성
>> ```java
>>static HashSet<String> h = new HashSet<String>();
>>```
  
>추가
>>```java
>>h.add(inputString);
>>```
>>
>>중복된 요소에관해서 h.add는 false를 반환하기때문에 판별이가능함<br/>h.add()? "잘들어갔어요!" : "이미있어요!";

>검색, 출력
>>```java
>>Iterator<String> iter = h.iterator();
>>while(iter.hasNext()) {
>>   System.out.println(iter.next());
>>}
>>```

>개수
>>```java
>>h.size();
>>```
